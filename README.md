# Lead Capture & CRM Automation

## My Role
Automation Developer (Make / CRM Integration Specialist)

---

## Project Overview
This project is an **automated lead management system built using Make**, integrating **Typeform with Pipedrive CRM**.

The goal was to eliminate manual data entry, prevent duplicates, and ensure structured lead processing in real time.

---

## Key Features
- Real-time capture of new Typeform submissions  
- Automatic detection of existing contacts and organizations  
- Deduplication logic to prevent duplicate records  
- Creation or updating of contacts and organizations in Pipedrive  
- Automatic lead generation and assignment  
- Routing logic and variable handling  
- Structured data processing  

---

## Scenario Architecture
The automation workflow was built in Make and includes:

- Trigger from Typeform submission  
- Data validation and normalization  
- Search for existing records in CRM  
- Conditional branching (create vs update logic)  
- Lead creation and assignment
